# Squad bootstrap

This directory holds repository-native memory and decisions for Squad-style workflows.

Current bootstrap content:

- `decisions.md` - initial architectural and governance decisions

After the repository is pushed and opened in a real environment:

1. run `squad init`
2. keep generated files that do not conflict with the project docs
3. preserve `decisions.md` as shared team memory

This repository expects Squad to accelerate repository work, not to act as an unrestricted VPS administrator.
